Summary

  • Html : 웹 페이지에서 화면에 표시되는 제목, 단락, 표 등과 같은 정보 구조뿐만 아니라 동영상, 링크 등의 정보 형식까지 표시하는 방법을 나타낸 언어
  • CSS : 마크업 언어인 HTML, XHTML, XML에서 색상, 폰트, 크기, 레이아웃 등으로 표시하는 방법을 나타낸 언어
  • Javascript : 웹에서 동적 처리를 위해 사용되는 객체지향 기반의 스크립트 언어
  • 웹서버 : 네트워크를 통해 클라이언트의 요청에 따라 정보를 제공하는 시스템
  • 클라이언트 : 네트워크를 통해 서버에게 정보를 제공 받는 응용 프로그램
  • WAS서버 : DB 조회나 다양한 로직 처리를 요구하는 동적인 컨텐츠를 제공하기 위해 만들어진 Web Application Server
